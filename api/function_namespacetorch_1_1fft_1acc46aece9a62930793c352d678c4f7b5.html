Function torch::fft::hfft¶

  • Defined in File fft.h

Function Documentation¶

Tensor torch::fft::hfft(const Tensor &self, c10::optional<int64_t> n = c10::nullopt, int64_t dim = -1, c10::optional<std::string> norm = c10::nullopt)¶

Computes the 1 dimensional FFT of a onesided Hermitian signal.

The input represents a Hermitian symmetric time domain signal. The returned Fourier domain representation of such a signal is a real-valued. See https://pytorch.org/docs/master/fft.html#torch.fft.hfft

Example: 

auto t = torch::randn(128 / 2 + 1, torch::kComplexDouble);
auto T = torch::fft::hfft(t, /*n=*/128);
assert(T.is_floating_point() && T.numel() == 128);
